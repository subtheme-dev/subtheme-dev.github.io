Sublate
Themes are built with Sublate, a high-level python toolkit for building templates. By using templates, colors can be defined with a single configuration and provide a more consistent user interface across editors.


How can I contribute?
Contributions are welcome with GitHub. Currently, the focus is porting popular themes to JetBrains IDEs. Support for additional editors will be added in the future. Any help is appreciated!
